使用GitHub做博客图床

Mar 31, 2017 | Hits
  1. 选择一个本地目录，创建一个README.md,Git Shell 进入该目录。
  2. git init.
  3. git add .
  4. git commit -m “first commit”
  5. git remote add origin git@github.com:your_github_username/projectname.jit
  6. git push -u origin master

博客建成后面临的第一个问题就是图床。百度一考查，又拍云、七牛等都不错。但是基本都是收费的。想到GitHub空间是完全免费的，就打起了GitHub的主意。实践了一下发现非常方便。

在GitHub上直接建一个repository，本地把图片分类存放好后设置为git仓库，push到GitHub上即可。图片地址非常规律，插入图片时灰常方便。
